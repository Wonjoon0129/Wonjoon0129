>
> 参与采购中台系统的改造与维护，负责与 MRO 系统、SCM 系统的集成。

- **模块开发**: 负责**招标、询价**等核心模块的业务需求改造与功能实现。

- **主要成果**: 实习期间独立负责 **7 个功能需求** 与 **3 个线上问题修复**，代码均**一次性通过**评审与测试。
